# 自定义API管理 v2.2.0

## 🚀 新增功能

### 1. JSON路径解析（核心升级）
支持自定义返回提取路径，例如：
data.url
result[0].img

### 2. API在线调试
指令：
/调试api weather 北京

返回完整接口响应（用于排错）

### 3. 防刷限流系统
- 每用户调用冷却时间
- 防止接口被刷爆

### 4. 智能解析增强
支持：
- JSON / TXT / 纯URL
- 自动识别图片/音频/视频

---

## 🧠 使用方式

/查api
/Rapi 指令 参数
/调试api 指令 参数

## ⚙️ 示例

/Rapi weather 北京
/调试api weather 北京
